Class: Core

enioka.ij. Core

Core module for the matrix library. This is the central node for all components. This is from this component that the matrix is displayed. It will ensure good working with each dependency. It holds, dataprovider, html renderer, and controller. Moreover, it contains two data handler which will ensure good understanding over data (for rows and columns).

To ensure good working, you have to user either our components, or yours. You can check in our documentations about APIs for each component.

This will also contain cache for data in a relativly near future
